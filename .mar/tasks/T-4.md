go-store's board puts "Under consideration" as the FIRST column. mar's
`column add` only supports `--after`, so a column cannot be inserted before the
first one (it lands last); reproducing the order required deleting and
re-adding all columns. Add `column add --before NAME` (or `--first`) and a
`column move` to reorder existing columns.
