Found while reproducing readcube/go-store docs in mar. go-store's 25 docs are
full HTML reports; mar only stores Markdown and has no import path, so each doc
had to be hand-converted. Add `mar doc import <file.html>` (or `--from-html`)
that converts HTML to mar Markdown + directives, so existing HTML doc sets can
be ingested without manual transcription.
