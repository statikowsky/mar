Allow editing a card (title, body) directly in the board/modal UI — edits
should be **inline** (edit in place, not a separate page/form), saved via an
endpoint.

**Prerequisite:** write a design doc (MAR `doc create --type design`) and get
it approved first — covers the inline edit interaction, which fields are
editable, the save endpoint, and live-reload behavior.

Pairs with card creation.
