go-store docs and tasks carry specific historical dates (2026-05-26 ...). mar
always stamps now(), so reproducing a dated archive loses the real dates. Add an
optional `--date`/`--updated` flag (or import-time preservation) for docs and
tasks so historical timelines survive a migration.
